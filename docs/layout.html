Loading screen

You can add a loading screen on your website. You just have to :

  • Add "loading" class on the body tag. 
    <body class="loading">...</body>
  • Add "loading" section in the template : 
    
    
    
    
    Loading
  • Add JS code : 
    $(window).load(function(){
    if($('#loading').length > 0) {
        $('#loading').delay(200).fadeOut(400);
        $('body').removeClass('loading');
    }
});

Scroll top button

You can easily add a scroll top button on your website. You just have to add this line to your main JS file. It will create the button automatically in your DOM and add events on it.

new ScrollTop();
