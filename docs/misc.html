Shortcuts

Margins & paddings generator

5px by 5px you can use css class to add margins or paddings.
Usable beetween 0 and 100px. You can combine them.

.mXX == margin: XXpx;
.mtXX == margin-top: XXpx;
.mrXX == margin-right: XXpx;
.mbXX == margin-bottom: XXpx;
.mlXX == margin-left: XXpx;

.pXX == padding: XXpx;
.ptXX == padding-top: XXpx;
.prXX == padding-right: XXpx;
.pbXX == padding-bottom: XXpx;
.plXX == padding-left: XXpx;
